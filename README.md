# EyepetizerMVPKotlin
## 模仿开眼视频的kotlin项目，采用单Activity+Fragment的设计，利用dagger优雅的实现mvp架构

##### 1.网络缓存+断网，出错自动重连（10次）  
  
##### 2。单Activity+多fragment仿照知乎，优雅的处理生命周期问题
  
##### 3.视频播放采用当前流行的GSYVideoPlayer框架

##### 4.后期会加入检测glide加载图片的进度显示
